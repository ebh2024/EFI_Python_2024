# Sistema de Gestión de Equipos

## Descripción

Este proyecto es un sistema de gestión de equipos, que permite administrar equipos, modelos, marcas, fabricantes, características, stock, proveedores y accesorios.

## Características

- CRUD (Crear, Leer, Actualizar, Eliminar) para cada entidad:
  - Equipo
  - Modelo
  - Marca
  - Fabricante
  - Característica
  - Stock
  - Proveedor
  - Accesorio

## Requisitos

- Python 3.x
- Flask
- Flask-WTF
- Flask-SQLAlchemy
- Flask-Migrate

## Instalación

1. Clona el repositorio:

   https://github.com/ebh2024/EFI_Python_2024.git
   cd EFI_Python_2024
   
2. Crea y activa un entorno virtual:   
     python -m venv .venv
     source .venv/bin/activate  # En Windows usa: .\.venv\Scripts\Activate

3. Instala las dependencias:
      pip install -r requirements.txt
